# Smartly Bridge Migration Progress

> 日期：2026-06-30  
> 分支：`dev`  
> 目標：將 `main` 上的既有 entity-based 架構逐步遷移到六邊形架構與 logical-device / capability-based contract。  
> 原則：所有 runtime 行為變更以 TDD slice 推進，並保留 legacy endpoint 相容性。
